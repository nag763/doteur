Since the version 0.5.0, doteur is shipped in several packages, for more information check the features section

All the downloads available below are for x86_64 platforms.

See the changelog if you need further details about the versions mentionned below.

If you have cargo installed, use :

cargo install doteur --all-features

For Windows users, don’t forget to check that you match the prequisites

For Linux users, don’t forget to check that you match the prerequisites.
